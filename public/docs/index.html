OpenFRBR Documentation

OpenFRBR is a Ruby on Rails application. To try it out you will need to have Ruby on Rails installed on your computer. You will also need SQLite version 3 and RubyGems. Your operating system probably has a way of using packages to make this really easy but you'll have to take of it all yourself.

Once you have that all ready, you will need to add has_many_polymorphs. 

$ sudo gem install has_many_polymorphs

Download openfrbr-2.1.tar.gz however you like to download files. 

$ tar xzpvf openfrbr-2.1.tar.gz
$ cd openfrbr-2.1
$ rake db:migrate
$ rake db:fixtures:load
$ ./script/server

Then follow the instructions to load up http://localhost:3000 in your web browser. If all went well, you should be able to use OpenFRBR locally.

OpenFRBR will be available through Git soon.
